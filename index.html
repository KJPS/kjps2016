KJPS 2016

KJPS 2016 nomtenē tēma būs tīmekļa spēļu izstrāde. Mērķis būs sadalīties komandās pa 2 un katrai komandai izveidot tīmekļa spēli 2 dienu laikā.

Izmantotās tehnoloģijas

  • JavaScript - valoda, kuru izmanto, lai programmētu dažādas darbības un loģiku, ieskaitot zīmēšanu. Vairāk lasīt par JavaScript šeit.
  • Pixi.JS - JavaScript bibliotēka, kuru izmanto, lai vienkāršāk zīmētu vai taisītu spēles. Vairāk lasīt par Pixi.JS šeit.
  • Simry.JS - PIXI.JS bāzēta bibliotēka. Vairāk lasīt par Simry.JS šeit.

Projekta izstrādes veidi

Katrai komandai (vai personai) ir iespēja izstrādāt tīmekļa spēli vienā no sekojošiem veidiem.

  1. Simry.JS - Bibliotēka iesācējiem bāzēta uz Pixi.JS. Šo izmantojot var mierīgi izmantot arī Pixi.JS.
  2. Pixi.JS - Vienkārša JavaScript grafiskā bibliotēka.
  3. Cita JavaScript bibliotēka - tiem, kam ir pārliecība par savām spējām un zināšanas par citām tehnoloģijām, lai izveidotu tīmekļa spēli, var izmantot tās.
  4. HTML - ja ļooooti gribās veidot spēli HTML. Bet ieteicams pieturēties pie JavaScripta un WebGL/Canvas tehnoloģijām.
